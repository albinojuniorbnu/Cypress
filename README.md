Passo a passo para configurar o projeto

Instalar o node js disponivel em: https://nodejs.org/en/

Acessar o diretório root do projeto (que tem a pasta Cypress), instalar as dependências e depois abrir o Cypress Runner, para isso seguir os passos abaixo:

1: npm install Cypress

2: .\node_modules\.bin\cypress open

Na janela do CY selecionar a opção Electron e executar através da opção Run all specs
